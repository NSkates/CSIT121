Reflections...

Lab 2 Reflection

For the second lab, I was able to create my first website from scratch! I was able to enforce proper HTML structure and use semantic tags effectively. My biggest takeaway from this lab that I learned while working on it was how to make a website print out tags. While working on this lab, when I tried to list the unique image and break tags, the brackets wouldn't show up on the website when I just typed "<img>". I later learned that in order to make that happen, you must include an "and" symbol (&), following "lt" and another "and" symbol (&) following "gt;". This was a very important lesson for me to learn because without it, the website would not display the tags properly!

Lab 3 Reflection

As for this latest third lab, I utilized the same website as before to help me apply what I learned about CSS. Specifically, external CSS. My biggest takeaway from this lab was honestly being careful on what fonts and colors I chose. I found it very easy to type what I wanted, but choosing what I wanted to put on here was a different story. I had to make sure that the colors I chose for the background and text were not too bright or too dull, and that the font I chose was easy to read.

Lab 5 Reflection

For this fifth lab, I really focused on ensuring my website would be interactive and responsive in different screen sizes. My desktop I code on is a 26 inch monitor, so I noticed when I set an internal anchor, there would barely be any scroll to the connecting blog post and I felt like that could be confusing to users (I'd like to also note that this wouldn't be an issue for mobile users). This made me remember when I looked through the psuedo lessons and I figured maybe that'll help make my website less confusing. I asked ChatGPT if it's possible to have the corresponding heading change color when a user clicks on the internal anchor within the navigation. My response was a "yes" and I was shown how to incorporate a target psuedo class in my CSS to make that happen. I thought that was a really good addition to make into this webpage because it makes the indication clearer for those with bigger screens (and adds more fun to those on smaller screens).

Here are the Most Important Things to Know When Creating a Website...

  1. Tags Must Have Open and Closing Brackets
  2. Attribute Tags linking to other page(s) must be enclosed in double quotes
  3. HTML's structure is like a skeleton!

  • <br>
